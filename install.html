Jak zainstalować paczke ?

  1. Wpisz uruchom w wyszukiwarke windowsa.
  2. Wpisz %appdata%
  3. Otwórz folder .minecraft
  4. Wrzuć tam wszystkie pliki z archiwum
  5. Otwórz swój launcher minecraft
  6. Zainstaluj wersje 1.16.5
  7. Kliknij New profile
  8. W zakładce use version wybierz release 1.16.5-forge
  9. Zaznacz JVM Arguments i zmień -Xmx1G na -Xmx4G lub -Xmx8G jeśli masz 16GB ramu
  10. Kliknij Save Profile
  11. Od teraz możesz grać na paczce EmpireCraft
